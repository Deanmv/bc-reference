Email campaign layout


{tag_addtofavorites,add_Image, remove_Image}Add item to favorites list. Optionally customize to display your own custom image for adding and removing.
{tag_anniversarytitle}Title of anniversary date relevant to each recipient (use with Anniversary 1-5campaigns only)
{tag_deliverydate}Outputs the date on which the campaign will be sent. Can be used in the campaign subject
{tag_recipientPasswordResetLink}Hyperlinks to the password reset form.
{tag_recipientemail}Email address of the recipient
{tag_recipientFirstName}First name of the recipient (replaces with 'Friend' if first name is empty)
{tag_recipientLastName}Last name of the recipient
{tag_recipientname}Full name of the recipient (replaces with 'Friend' if full name is empty)
{tag_recipientpassword}Password of the recipient. Use to login to a Secure Zone - DEPRECATED - For security purposes, passwords are no longer sent
{tag_recipientPasswordResetLink}Link for user to reset/update existing password
{tag_recipientusername}Username of the recipient. Use to login to a Secure Zone
{tag_recipientExternalID}ExternalID of the recipient (specified when importing contacts only).
{tag_siteurl}URL to your public website
{tag_siteUrlEncoded}encoded URL to your public website
{tag_tellafriend}Tell a friend about this item. NOTE: you can customise the link text that tag renders using this method {tag_tellafriend,YourCustomText}
{tag_unsubscribe}Hyperlink to enable recipients to opt-out of all future newsletters. This tag is mandatory. NOTE: you can customise the link text that tag renders using this method {tag_unsubscribe,YourCustomText}
{tag_unsubscribelist}Hyperlink to enable recipients to unsubscribe from the list associated with this campaign. The customer will continue to receive other newsletters. NOTE: you can customise the link text that tag renders using this method {tag_unsubscribelist,YourCustomText}
{tag_unsubscribelisturl}URL to enable you to form your own unsubscribe link associated with this campaign.
{tag_viewinbrowser}View this email in browser hyperlink. NOTE: you can customise the link text that tag renders using this method {tag_viewinbrowser,YourCustomText} Also note, if you customize the text from "Unsubscribe" to something else, the "Unsubscribe" link will automatically be appended to the bottom of your newsletter
{tag_viewinbrowserurl}View this email in browser URL
